안녕하세요,

CQ 4주 체험단에 초대합니다.

CQ는 AI와 나눈 대화와 실험을 누적해 개인 외장 두뇌로 발전시키는 도구입니다. 실험 간 연결(derives_from)을 선언하면 지식이 체인으로 쌓이고, 나중에 맥락을 다시 불러올 수 있습니다.

설치 (원라인) ​

sh
curl -fsSL https://raw.githubusercontent.com/PlayIdea-Lab/cq/main/install.sh | sh

설치 후 cq auth login 으로 로그인하고, 아래 초대 코드를 입력해 주세요.

초대 코드: [초대 코드]

체험 기간 ​

4주: [날짜] ~ [날짜]

참여 방식 ​

  • 평소처럼 GPU 실험을 진행하시면 됩니다.
  • 별도 안내나 과제는 없습니다.
  • nudge 없음, 자연 사용 — 저희는 어떤 실험을 해야 한다거나 다음 단계를 제안하지 않습니다. 체험자분이 자연스럽게 사용하는 패턴을 관찰하는 것이 목적입니다.

데이터 격리 ​

체험자분의 실험 데이터는 다른 체험자와 완전히 격리됩니다 (RLS 기반). 다른 분의 데이터를 볼 수 없고, 반대로 본인 데이터도 타인에게 노출되지 않습니다.
