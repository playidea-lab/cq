## How It Works

```
 You say              CQ does                    Result
──────────────────────────────────────────────────────────
 "build this"         /pi  → brainstorm + research   idea.md
 "go"                 /c4-plan → tasks + review       plan
 ⏳                   /c4-run  → parallel workers     code + tests
 ☕                   /c4-finish → polish + verify    done
```
