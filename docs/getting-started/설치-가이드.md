# C4 설치 가이드

이 문서는 C4를 설치하고 Claude Code에서 사용할 수 있도록 설정하는 방법을 안내합니다.

## 사전 요구사항

### 필수
- **Python 3.12+**: C4는 Python 3.12 이상이 필요합니다
- **uv**: Python 패키지 관리자 ([설치 가이드](https://docs.astral.sh/uv/getting-started/installation/))
- **Claude Code**: Anthropic의 AI 코딩 어시스턴트

### 선택
- **Git**: 버전 관리 (프로젝트에서 사용 시 필요)

## 빠른 설치 (권장)

터미널에서 한 줄로 설치:

```bash
curl -LsSf https://git.pilab.co.kr/pi/c4/-/raw/main/install-remote.sh | sh
```

설치 스크립트가 자동으로 다음 작업을 수행합니다:
- C4를 `~/.c4`에 클론
- `uv sync`로 의존성 설치
- `~/.claude/commands/`에 슬래시 명령어 복사
- `~/.claude.json`에 MCP 서버 설정 추가

설치 위치를 변경하려면:
```bash
curl -LsSf https://git.pilab.co.kr/pi/c4/-/raw/main/install-remote.sh | sh -s -- --dir ~/tools/c4
```

### 대안: Clone & Install

```bash
git clone https://git.pilab.co.kr/pi/c4.git
cd c4
./install.sh
```

### Claude Code 재시작

설정을 적용하려면 Claude Code를 **완전히 종료 후 다시 시작**합니다.

### 설치 확인

Claude Code에서 다음 명령을 실행합니다:

```
/c4-status
```

성공 시 다음과 비슷한 메시지가 표시됩니다:
```
C4 not initialized in this project.
Use /c4-init to initialize.
```

## 수동 설치 (고급)

자동 스크립트 대신 직접 설치하려면:

### 1. 의존성 설치

```bash
cd c4
uv sync
```

### 2. 슬래시 명령어 복사

```bash
mkdir -p ~/.claude/commands
cp .claude/commands/c4-*.md ~/.claude/commands/
```

### 3. MCP 서버 등록

`~/.claude.json` 파일을 편집하여 `mcpServers`에 추가:

```json
{
  "mcpServers": {
    "c4": {
      "command": "uv",
      "args": ["--directory", "/path/to/c4", "run", "python", "-m", "c4.mcp_server"]
    }
  }
}
```

> **중요**: `/path/to/c4`를 실제 C4 설치 경로로 변경하세요.

### 4. Claude Code 재시작

설정을 적용하려면 Claude Code를 재시작합니다.

## 프로젝트별 설정 (선택)

특정 프로젝트에서만 C4를 사용하려면 프로젝트 루트에 `.claude/settings.json`을 생성합니다:

```json
{
  "mcpServers": {
    "c4": {
      "command": "uv",
      "args": ["--directory", "/path/to/c4", "run", "python", "-m", "c4.mcp_server"]
    }
  }
}
```

## 문제 해결

### "MCP server not found" 오류

1. C4 경로가 올바른지 확인
2. `uv sync`로 의존성 설치 확인
3. Claude Code 재시작

### "Python not found" 오류

1. Python 3.12+ 설치 확인: `python --version`
2. uv 설치 확인: `uv --version`

### 슬래시 명령이 보이지 않음

1. `~/.claude/commands/` 폴더 확인:
   ```bash
   ls ~/.claude/commands/c4-*.md
   ```
2. 없으면 다시 복사:
   ```bash
   cp /path/to/c4/.claude/commands/c4-*.md ~/.claude/commands/
   ```
3. Claude Code 완전 재시작

### MCP 도구가 작동하지 않음

1. `~/.claude.json`에 c4 MCP 서버가 등록되어 있는지 확인
2. MCP 서버 경로가 올바른지 확인
3. Claude Code 완전 재시작

## 업데이트

C4를 최신 버전으로 업데이트하려면:

```bash
cd /path/to/c4
git pull
./install.sh
```

그 다음 Claude Code를 재시작합니다.

---

## 제거

C4를 제거하려면:

```bash
# 1. 슬래시 명령어 삭제
rm ~/.claude/commands/c4-*.md

# 2. ~/.claude.json에서 "c4" MCP 서버 항목 삭제

# 3. (선택) C4 폴더 삭제
rm -rf /path/to/c4
```
