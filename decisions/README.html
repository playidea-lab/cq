docs/decisions — 결정 기록 (ADR) ​

중요한 아키텍처·제품 결정을 ADR(Architecture Decision Record) 형식으로 보존합니다.

위치 안내 ​

저장소용도
docs/decisions/공식 ADR 문서 (이 디렉토리)
.c4/ideas/런타임 아이디어·탐색 노트 (에이전트가 작성, git tracked)
docs/ir/투자·내부 보고서 (비공개 참고 자료)
.c4/designs/구현 설계 초안 (에이전트가 작성)

ADR 작성 규약 ​

파일명: ADR-XXX-<kebab-title>.md

markdown
## ADR-XXX: [제목]
- **상태**: 제안 / 승인 / 폐기
- **맥락**: 왜 이 결정이 필요한가
- **결정**: 무엇을 선택했는가
- **근거**: 왜 이것을 선택했는가
- **대안**: 고려했지만 선택하지 않은 것
- **결과**: 이 결정으로 인해 변하는 것

이전 결정 기록 탐색 ​

  • .c4/ideas/*.md — 탐색 단계 노트 및 방향 결정
  • docs/specs/ — 확정된 스펙 문서
  • CHANGELOG.md — 릴리즈별 결정 요약
