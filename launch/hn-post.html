Title ​

Show HN: CQ – AI agent orchestration with persistent memory and verifiable experiments

Body (1 paragraph) ​

We've been building CQ for the past year as the external brain for AI agents: every conversation across Claude Code, ChatGPT, Cursor, and Gemini accumulates into a searchable knowledge base, and every ML experiment links to a verifiable URL with metric, command, and git SHA. The core is a Go MCP server that runs locally (stdio) with a Python sidecar for document parsing and persona learning. Key pieces: a task orchestrator that spawns parallel Claude workers for large refactors, an end-to-end relay for NAT-traversed GPU workers, and a 3-layer wire validation system we built after shipping v1.60.0 with four silent wire bugs that passed 29 code reviews. Pro tier ($36/yr Early Bird, first 1,000 users) adds cross-project memory, cross-platform sync, and autonomous research loops on remote GPUs. Feedback welcome — especially on the Evidence Web URL scheme and the persona learning approach.
